== Installation ==

1. Upload the theme files to `/wp-content/themes/kr-theme/` directory
2. Activate the theme through 'Appearance > Themes' menu in WordPress
3. Install the recommended KR Toolkit plugin for demo import and additional features
4. Go to Appearance > Customize to configure theme options

== Frequently Asked Questions ==

= Does this theme work with page builders? =

Yes! KR Theme is fully compatible with Elementor and other popular page builders.

= Is this theme SEO friendly? =

Absolutely! KR Theme is built with SEO best practices including schema.org markup, semantic HTML, and optimized performance.

= Can I use this theme for e-commerce? =

Yes! KR Theme has full WooCommerce integration for building online stores.

= Does it support RTL languages? =

Yes, KR Theme includes RTL language support out of the box.

= How do I achieve 100/100 PageSpeed scores? =

KR Theme is optimized for speed by default. For best results:
- Use optimized images (WebP format recommended)
- Enable Gzip compression on your server
- Use a reliable hosting provider
- Install the KR Toolkit companion plugin
